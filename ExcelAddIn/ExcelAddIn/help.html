FinLite Excel Add-In Help

Introduction

FinLite is a VSTO-based Excel add-in that equips with LLM-powered data analysis and question answering.

Using the Add-In

  • Select Cells: Select a range in Excel to use as specific context for the chat.
  • No Cells Selected: All sheets will be used as context for the chat.
  • Ask Questions: Enter your prompt and click "Enter" button to query the model.
  • Verbosity: Use the dropdown to choose Concise or Detailed responses.
