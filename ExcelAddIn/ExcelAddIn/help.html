FinLite Assistant Add-In Help

Introduction

FinLite is a VSTO-based Excel add-in that equips with LLM-powered data analysis and question answering.

Using the Add-In

Core Features

  • Cell Selection: Select a range in Excel to use as specific context for your questions. The selected data will appear in the "Selection preview" panel, or leave unselected to use all sheets.
  • Ask Questions: Type your question in the prompt box and press Enter or click the Send button. Use Ctrl+Enter to add new lines in your question.
  • Chat History: All questions and answers are saved and displayed in the History panel. Previous conversations can be accessed and reloaded via the History button.

Interface Controls

  • Verbosity Dropdown: Choose between Concise or Detailed responses to control answer length.
  • New Chat Button (+ icon): Start a fresh conversation, clearing the chat history.
  • History Button (clock icon): View and reload previous chat sessions.
  • About Button (i icon): View information about the add-in and development team.
  • Help Button (? icon): Open this help documentation.
